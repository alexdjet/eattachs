## About
Утилита для сохранения вложений из писем (IMAP)

    Простое CLI-приложение на Go, которое подключается к почтовому серверу по IMAP, авторизуется с помощью переменных окружения, получает список не прочитанных писем, сохраняет вложения в директорию. Полезно для автоматизации обработки входящих писем.
    Эта программа — как робот, который заходит в твой email, ищет письма и автоматически сохраняет все вложения (файлы, которые к ним прикреплены) на компьютер.

## Возможности
- Отображение версии и времени сборки
- Подключение к IMAP-серверу по данным из .env
- Аутентификация по логину и паролю
- Получение списка писем с почтового ящика
- Сохранние прикрепленных файлов
- Поддержка кастомной директории для работы

## Требования
- Go 1.19+
- Доступ к почтовому ящику с включённым IMAP (например, Яндекс, Gmail и т.д.)


## Настройка

```env
IMAP_SERVER=imap.example.com:993
IMAP_USER=your_email@example.com
IMAP_PASS=your_password_or_app_token
FROM_EMAIL=filter_from@example.com
FROM_SUBJECT=фильтр по теме письма (опционально)
WORK_DIR=./downloads
```

## Сборка
Для сборки бинарного файла с указанием версии и времени:
```bash
# for unix
make build 

# for windows
make build/win
```

## Запуск
```bash
# for unix
./bin/eattachs
or 
# for windows
./bin/eattachs_win
```

## Безопасность
- Используйте токены приложений вместо паролей (особенно для Gmail, Яндекс и др.)
- Храните .env вне системы контроля версий
